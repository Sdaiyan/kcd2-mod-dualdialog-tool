# Dual Dialog

**Dual Dialog** 是一款适用于《天国：拯救 2》的 Mod 生成工具，可将游戏内文本同时以两种语言显示。无论你是想在母语环境下顺带强化第二外语，还是单纯享受双语沉浸体验，这款工具都能满足你的需求。

《天国：拯救 2》（KCD2）以丰富的对话和出色的配音著称。本工具正是借助这一优势，让你在沉浸故事的同时，顺带提升语言能力。

本项目托管于 [GitHub](https://github.com/SDxBacon/kcd2-mod-dualdialog-tool)。

> 📖 [English README](./README.md)

## 功能特性

- **双语同屏显示**：游戏内文本同时显示主语言与配对语言。
- **广泛的内容覆盖** — 支持 8 个双语化分类：
  - ✅ **对话** — 头顶对话、配音对话、剧情对话序列
  - ✅ **任务日志** — 任务名称与描述
  - ✅ **炼金及物品** — 物品名称、描述和配方文本
  - ✅ **技能与增益** — 技能名称与天赋描述
  - ✅ **成就** — 成就名称与描述
  - ✅ **庄园小游戏** — 小游戏相关 UI 文本
  - ⚠️ **菜单与百科** — 菜单标签和百科条目（可能存在显示异常）
  - ⚠️ **教程** — 含 UI 标记的教程文本（可能存在显示异常）
- **分类独立分隔符**：每个分类可独立配置 `\n`（游戏内换行）或 ` / `（行内斜杠）。
- **社区补丁支持**：生成前可为主语言或副语言加载一个或多个社区修复 XML 文件（如汉化修正包），补丁条目优先级高于原始 Pak 数据。
- **无需完整游戏安装**：只需包含语言 `.pak` 文件的 `Localization/` 文件夹即可运行。

## 使用方法

### 基础使用

1. **启动工具** — 运行 `Dual Dialog Tool.exe`。
2. **选择游戏文件夹** — 选择包含 `Localization/` 子文件夹（内有语言 `.pak` 文件）的目录。无需指向完整游戏安装目录。
3. **选择语言** — 选择游戏主语言和希望同屏显示的配对语言。
4. **导出** — 点击 **导出**，选择输出文件夹。工具将生成 `Dual Dialog.zip`。
5. **安装 Mod** — 将 `Dual Dialog.zip` 解压到 KCD2 的 `Mods` 文件夹（例如 `…\KingdomComeDeliverance2\Mods`）。如文件夹不存在，手动创建即可。
6. **启动游戏** — 进入游戏，享受双语文本体验！

### 高级设置

点击工具界面中的 **▸ 高级设置** 可访问以下选项：

- **内容分类** — 独立开启/关闭 8 个分类，并为每个分类设置自定义分隔符。
- **主 / 副语言补丁包** — 为任意一侧语言添加社区补丁 XML 文件。工具会实时预览每个补丁包含的条目数量。补丁条目将覆盖原始 Pak 数据后再进行双语合并。

## 推荐语言组合

KCD2 各语言包仅内置该语言所需的字体，加载字体包以外的文字将导致字符缺失。

| 主语言 | 配对语言 | 效果 |
|---|---|---|
| 拉丁语系（如英语） | 拉丁语系（如法语） | ✅ 正常显示 |
| 亚洲语系（如中文） | 拉丁语系（如英语） | ✅ 正常显示 |
| 亚洲语系（如中文） | 俄语 | ⚠ 部分西里尔字符缺失 |
| 拉丁语系 | 亚洲语系（如中文） | ❌ 大部分汉字无法显示 |
| 简体中文 | 日语 | ❌ 亚洲语系之间字体不共用 |
